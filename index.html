JavaScript (JS)

Javascript is a language used to make webpage interactive.

Operators

Operators in JS is used to perform arithmetics on numbers or varibles representing numbers.

Variables

Variables can be thought of as containers used to store information. They allow for a way to label data with a descriptive name.

Variable Naming Convention

A set of rules to name variables.

Functions

A function is a "subprogram" that can be called.

Methods

A method is a function which is a property of an object.
